# TabBar 图标说明

## 需要的图标文件

为了完整显示 TabBar，需要添加以下图标文件：

### 统计页面图标
- `statistics_default.png` - 未选中状态的统计图标（建议尺寸：81x81px）
- `statistics_selected.png` - 选中状态的统计图标（建议尺寸：81x81px）

### 账单页面图标
- `bill_default.png` - 未选中状态的账单图标（建议尺寸：81x81px）
- `bill_selected.png` - 选中状态的账单图标（建议尺寸：81x81px）

## 图标设计建议

- **统计图标**：可以使用图表、数据统计相关的图标
- **账单图标**：可以使用账单、列表、记录相关的图标
- **颜色**：未选中状态使用灰色（#7A7E83），选中状态使用主题色（#27ba9b）

## 临时方案

如果暂时没有图标，可以：
1. 使用在线图标生成工具创建
2. 使用现有的图标作为占位符
3. 使用纯色占位图片
